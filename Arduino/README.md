Um sistema utilizando arduino, uma placa de ensaio, 3 resistores, um led rgb e 3 potenciômetros.
Cada potenciômetro controla isoladamente uma cor do rgb, vermelho, azul e verde.
Utiliza variáveis e mapeamento.
